
---------- How to Login -----------------


For login passwords and work id you must look in Logins.json file.
This file placed in MarketServer/src folder.



----------- JSON simple jar -------------


If you have problems with JSON while loading project, set path to json-simple-1.1.1.jar
that located in same archive.


---------- JUnit 5 -------------

If you have problems with JUnit create path to JUnit5 library
